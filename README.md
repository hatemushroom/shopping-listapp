# 쇼핑 리스트 앱

브라우저에서 바로 실행되는 간단한 쇼핑 리스트 앱입니다. 별도 설치나 서버 없이 `index.html` 파일 하나로 동작합니다.

## 기능

- 항목 추가 / 삭제
- 항목 체크(완료) 표시
- 체크된 항목 일괄 삭제
- 총 개수 / 완료 개수 표시
- `localStorage`를 이용한 자동 저장 (새로고침해도 목록 유지)

## 사용 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

```bash
# 예: 기본 브라우저로 열기 (Windows)
start index.html
```

## 기술 스택

- HTML / CSS / JavaScript (바닐라, 별도 프레임워크 없음)
